# Visomaster-Fusion-Patches
A patch that adds stuff for Visomaster that developers refused and fixes "features" that they add. 

-------------------------------------------------------------------------------------------------------------

Check [releases](https://github.com/masterofobzene/Visomaster-Fusion-Patches/releases) for details on what these patches do.

-------------------------------------------------------------------------------------------------------------

This mod is rather for patient people that will not update everytime an update comes. 'N' is your friend here. **Every update will replace your patch with the vanilla version of the app**. Development is very unstable so you should be doing this anyway. I will do my best to update it when a significant update comes since I am using it, you can expect that to happen anyway. 
My personal advice is to make a backup before updating the app.


### INSTRUCTIONS:

1 - Download the "app.zip" from releases. MATCH YOUR VISOMASTER VERSION otherwise it will have unpredictable results.

2 - Unzip into ..\VisoMaster-Fusion\  replacing your current `app` folder.

3 - Replace all files.



### THIS IS NOT THE ENTIRE APP. YOU MUST INSTALL IT FIRST USING THE PORTABLE VERSION THEY HAVE. RIGHT CLICK ON [THIS LINK](https://github.com/VisoMasterFusion/VisoMaster-Fusion/blob/dev/Start_Portable.bat) -> SAVE AS.
